8. Server Action vs API Route

Server Action ("use server")

WHO calls it? → Your React app

  • Form submissions
  • Button click mutations
  • Revalidating data

API Route (route.ts)

WHO calls it? → External services

  • Stripe webhooks
  • Mobile apps
  • Public REST API

9. Server vs Client Components

Server Component (default)

  • Runs on server
  • NO JS sent to browser
  • CAN: fetch data, access DB, env vars
  • CANNOT: useState, useEffect, onClick

Client Component ("use client")

  • Runs in browser
  • JS included in bundle
  • CAN: hooks, events, browser APIs
  • CANNOT: access DB directly
